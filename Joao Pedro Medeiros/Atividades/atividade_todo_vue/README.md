# Atividade To-Do Vue 3

## Objetivo

Refatorar uma aplicação To-Do utilizando Vue 3.

---

## Recursos utilizados

- Vue 3
- v-model
- @click
- v-for
- Props
- Emits
- Componentes

---

## Funcionalidades

- Adicionar tarefa
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa

---

## Como executar

Instalar dependências:

```bash
npm install
```

Executar projeto:

```bash
npm run dev
```

Abrir o navegador no endereço:

```txt
http://localhost:5173/
```
